Faceprox

Proxy for rendering Facebook pages. Very much a proof of concept.

Path Description
/events/:eventid Render a Facebook event
/events/:eventid.json Get a json representation of a Facebook event
/events/:eventid.ics Get a ical representation of a Facebook event
/page/:pagename Render a list of events for a given page name
/events/:eventid.json Get a json representation of events for a given page name
/events/:eventid.ics Get a ical representation of events for a given page name
